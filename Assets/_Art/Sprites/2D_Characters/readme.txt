HOW TO INSTALL AND USE THE ASSETS

1) Import this package in Unity (Tab Assets > Import New Asset).
2) Open the Project window (Tab Window > Project).
3) Drag an asset from the project window to your scene. This will automatically create a game object with your 2d sprite included in it.
